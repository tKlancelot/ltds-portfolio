LTDS Design System

Un design system orienté code, modulaire et extensible.

Avec 6 ans d’expérience en développement front-end, je conçois des interfaces web robustes et centrées sur l’utilisateur. Mon approche met l’accent sur un code propre, maintenable et une intégration UI fluide.

  • atoms : avatars, alerts, inputs, links, icons
  • molecules : buttons, cards, forms, modals

Installez le Design System via npm et commencez à l’utiliser immédiatement. Vous pouvez importer la feuille de style complète ou seulement les parties qui vous intéressent grâce au tree-shaking.

npm install @tarik-leviathan/ltds@latest

Importer tout le Design System :

import '@tarik-leviathan/ltds/dist/ltds.css';

Ou seulement les modules dont vous avez besoin :

import '@tarik-leviathan/ltds/styles/core'; // reset, tokens de base import '@tarik-leviathan/ltds/styles/buttons'; // styles de boutons import '@tarik-leviathan/ltds/styles/utils'; // utilitaires
