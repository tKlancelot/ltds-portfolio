Utilitaires

Ensemble de classes prêtes à l’emploi pour gérer couleurs, bordures, textes, tailles et comportements courants. Inspiré des utility-first frameworks (ex. Tailwind) mais limité à notre Design System.

Utilitaires pour activer/désactiver ou ajuster l’épaisseur des bordures.

.u-border { border-style: solid; border-width: 1px; } .u-border-0 { border-width: 0; } .u-border-1 { border-width: 1px; } .u-border-2 { border-width: 2px; }

Classes pour appliquer rapidement des fonds colorés ou semi-transparents.

.u-bg-primary { background-color: var(--color-primary); } .u-bg-success { background-color: var(--color-success); } .u-bg-warning { background-color: var(--color-warning); } .u-bg-info { background-color: var(--color-info); } .u-bg-danger { background-color: var(--color-danger); } .u-bg-primary-opacity-10 { background-color: var(--color-primary-opacity-10); } .u-bg-success-opacity-10 { background-color: var(--color-success-opacity-10); }

Permet de changer la couleur du texte en fonction des tokens.

.u-text-primary { color: var(--color-text-primary); } .u-text-secondary { color: var(--color-text-secondary); } .u-text-danger { color: var(--color-text-danger); } .u-text-success { color: var(--color-text-success); } .u-text-warning { color: var(--color-text-warning); }

Helpers pour la casse, le style ou les tailles de texte.

.u-text-uppercase { text-transform: uppercase; } .u-text-capitalize { text-transform: capitalize; } .u-text-style-italic { font-style: italic; } .u-fs-1 { font-size: var(--font-size-1); } .u-fs-2 { font-size: var(--font-size-2); } .u-fs-3 { font-size: var(--font-size-3); }

Utilitaires généraux (overflow, z-index, object-fit, etc.).

.u-overflow-hidden { overflow: hidden; } .u-overflow-y-auto { overflow-y: auto; } .u-z-i-9 { z-index: 9; } .u-object-fit-cover { object-fit: cover; }

Arrondis standardisés et hauteurs de ligne basées sur nos tokens.

.u-radii-2 { border-radius: var(--radii-2); } .u-radii-3 { border-radius: var(--radii-3); } .u-radius-round { border-radius: var(--radii-round); } .u-lh-1 { line-height: var(--lh-1); } .u-lh-2 { line-height: var(--lh-2); } .u-lh-prc-3 { line-height: 150%; }
