Utilitaires

Ensemble de classes prêtes à l’emploi pour gérer couleurs, bordures, textes, tailles et comportements courants. Inspiré des utility-first frameworks (ex. Tailwind) mais limité à notre Design System.

Largeurs standards et couleurs basées sur les tokens (dont variantes atténuées -20).

/* Largeur */ .u-border-0 { border-width: 0; } .u-border-1 { border-width: 1px; } .u-border-2 { border-width: 2px; } .u-border { border: 1px solid var(--color-border); } /* Couleurs génériques */ .u-border-default { border-color: var(--color-border); } .u-border-strong { border-color: var(--color-border-strong); } .u-border-inverse { border-color: var(--color-border-inverse); } /* Sémantiques pleines */ .u-border-primary { border-color: var(--color-border-primary); } .u-border-success { border-color: var(--color-border-success); } .u-border-warning { border-color: var(--color-border-warning); } .u-border-info { border-color: var(--color-border-info); } .u-border-danger { border-color: var(--color-border-danger); } /* Sémantiques atténuées (opacity-20) */ .u-border-primary-20 { border-color: var(--color-border-primary-20); } .u-border-success-20 { border-color: var(--color-border-success-20); } .u-border-warning-20 { border-color: var(--color-border-warning-20); } .u-border-info-20 { border-color: var(--color-border-info-20); } .u-border-danger-20 { border-color: var(--color-border-danger-20); }

Exemple : bordure de modal légère → border:1px solid var(--color-border-primary-20) ou classe u-border u-border-primary-20.

Classes pour appliquer rapidement des fonds colorés ou semi‑transparents.

/* Solides */ .u-bg-primary { background-color: var(--color-primary); } .u-bg-success { background-color: var(--color-success); } .u-bg-warning { background-color: var(--color-warning); } .u-bg-info { background-color: var(--color-info); } .u-bg-danger { background-color: var(--color-danger); } /* Opacités */ .u-bg-primary-10 { background-color: var(--color-primary-opacity-10); } .u-bg-primary-20 { background-color: var(--color-primary-opacity-20); } .u-bg-primary-40 { background-color: var(--color-primary-opacity-40); } .u-bg-success-10 { background-color: var(--color-success-opacity-10); } .u-bg-success-20 { background-color: var(--color-success-opacity-20); } .u-bg-warning-10 { background-color: var(--color-warning-opacity-10); } .u-bg-warning-20 { background-color: var(--color-warning-opacity-20); } .u-bg-info-10 { background-color: var(--color-info-opacity-10); } .u-bg-info-20 { background-color: var(--color-info-opacity-20); } .u-bg-danger-10 { background-color: var(--color-danger-opacity-10); } .u-bg-danger-20 { background-color: var(--color-danger-opacity-20); }

Couleurs de texte alignées sur les tokens (primaire, secondaires et sémantiques).

.u-text-primary { color: var(--color-text-primary); } .u-text-secondary { color: var(--color-text-secondary); } .u-text-tertiary { color: var(--color-text-tertiary); } .u-text-inverted { color: var(--color-text-inverted); } .u-text-translucent { color: var(--color-text-translucent); } .u-text-success { color: var(--color-text-success); } .u-text-warning { color: var(--color-text-warning); } .u-text-info { color: var(--color-text-info); } .u-text-danger { color: var(--color-text-danger); } /* Gradient basé sur --text-gradient */ .u-text-gradient-primary { background: var(--text-gradient); -webkit-background-clip:text; -webkit-text-fill-color:transparent; }

Helpers de casse/style, tailles tokenisées et alignements (y compris responsives).

/* Casse & style */ .u-text-uppercase { text-transform: uppercase; } .u-text-capitalize { text-transform: capitalize; } .u-text-style-italic { font-style: italic; } /* Poids (extraits) */ .u-fw-normal { font-weight: 400; } .u-fw-medium { font-weight: 500; } .u-fw-semibold { font-weight: 600; } .u-fw-bold { font-weight: 700; } /* Tailles (extraits) */ .u-fs-1 { font-size: var(--tfs-1); } .u-fs-2 { font-size: var(--tfs-2); } .u-fs-3 { font-size: var(--tfs-3); } /* Alignements */ .u-text-left { text-align: left; } .u-text-center { text-align: center; } .u-text-right { text-align: right; } .u-text-justify { text-align: justify; } /* Alignements responsives (≤ breakpoint) */ /* Exemple mobile : */ .u-text-mob-center { text-align: center; } /* Idem dispo pour sm/tab/note/lap/desk */

Exemple : <p class="u-text-left u-text-mob-center"> → gauche par défaut, centré en mobile (≤768px).

Contrôler l’affichage selon le breakpoint (pattern .u-display-{bp}-{val}).

/* Valeurs : none | block | flex | inline | inline-block */ /* Exemples */ .u-display-mob-none { display: none !important; } .u-display-mob-block { display: block !important; } .u-display-desk-none { display: none !important; }
