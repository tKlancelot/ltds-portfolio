Grids

Système de grille minimal basé sur CSS Grid : colonnes 1→12, spans ligne/colonne, alignements utiles et variantes responsives (max-width) : desk, lap, note, tab, mob, smob. Idéal pour structurer des pages et des cartes sans surcouche CSS.

Active CSS Grid sur un conteneur, puis choisis le nombre de colonnes.

.u-grid { display: grid; } .u-grid-12 { grid-template-columns: repeat(12, minmax(0, 1fr)); } .u-grid-cols-3 { grid-template-columns: repeat(3, minmax(0, 1fr)); } /* 1 → 12 */

Étendre un item sur N colonnes (1→12) ou N lignes (1→3).

.u-col-span-6 { grid-column: span 6 / span 6; } /* 1 → 12 */ .u-row-span-2 { grid-row: span 2 / span 2; } /* 1 → 3 */ .u-col-auto { grid-column: auto; } .u-row-auto { grid-row: auto; }

Aligner les items et le contenu du grid selon vos besoins.

.u-items-start { align-items: start; } .u-items-center { align-items: center; } .u-items-end { align-items: end; } .u-justify-start { justify-items: start; } .u-justify-center { justify-items: center; } .u-justify-end { justify-items: end; } .u-content-start { align-content: start; } .u-content-center { align-content: center; } .u-content-end { align-content: end; } .u-place-center { place-items: center; } .u-place-self-start { place-self: start; } .u-place-self-center { place-self: center; } .u-place-self-end { place-self: end; }

Variantes par palier : desk (≤1919), lap (≤1439), note (≤1199), tab (≤1024), mob (≤768), smob (≤414).
On préfixe les classes : .u-{bp}-grid-cols-N, .u-{bp}-col-span-N, etc.

/* Exemple */ .u-mob-grid-cols-2 { grid-template-columns: repeat(2, minmax(0, 1fr)); } .u-mob-col-span-2 { grid-column: span 2 / span 2; } .u-tab-grid-cols-3 { grid-template-columns: repeat(3, minmax(0, 1fr)); }

Astuce : combinez u-grid-12 + u-mob-grid-cols-2 pour un layout fluide.

  • Utilisez u-grid-12 comme base “maillage” et des u-col-span-* pour rythmer les blocs.
  • Pour la lisibilité, préférez peu de paliers (ex. tab et mob) plutôt que tous à la fois.
  • Composez avec vos tokens : style="gap: var(--gap-4)" ou via une classe utilitaire si prévue.
