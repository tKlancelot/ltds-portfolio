Colors

La palette LTDS expose des tokens sources (brand, UI), puis un remap officiel vers des tokens consommables (--color-*, --surface-*, gradients, borders…). Les [data-scheme] permettent de surcharger proprement le branding et les fonds.

Fonds plus contrastés (esprit GitHub Dark). Utilisés tels quels ou via alias --surface-*.

:root { --color-ui-white: #fff; --color-ui-black: #000; --color-ui-white-opacity-92: rgba(255, 255, 255, 0.92); --color-ui-white-opacity-80: rgba(255, 255, 255, 0.80); --color-ui-white-opacity-60: rgba(255, 255, 255, 0.60); --color-ui-white-opacity-40: rgba(255, 255, 255, 0.40); --color-ui-white-opacity-20: rgba(255, 255, 255, 0.20); /* Fonds par défaut */ --color-background: #0d1117; /* page */ --color-container: #161b22; /* surfaces */ --color-overlay: #0c162d; /* overlays */ }

Conseil : vos composants peuvent pointer sur --surface-page|container|overlay (voir Remap).

Hiérarchie de lisibilité basée sur les opacités de blanc + sémantiques.

:root { --color-text-primary: var(--color-ui-white-opacity-92); --color-text-secondary: var(--color-ui-white-opacity-60); --color-text-tertiary: var(--color-ui-white-opacity-40); --color-text-inverted: var(--color-ui-black); --color-text-translucent: var(--color-ui-white-opacity-20); --color-text-danger: var(--color-danger); --color-text-success: var(--color-success); --color-text-info: var(--color-info); --color-text-warning: var(--color-warning); }

Tokens sobre par défaut, + variantes sémantiques alignées sur le brand (thème-aware).

:root { --color-border: rgba(255,255,255,0.12); --color-border-strong: rgba(255,255,255,0.25); --color-border-inverse:rgba(0,0,0,0.25); /* sémantiques (remappées plus bas pour suivre les schemes) */ --color-border-primary: var(--brand-primary); --color-border-primary-20: var(--brand-primary-opacity-20); }

Couleurs de marque et dégradés source, remappés ensuite en tokens de consommation.

:root { --brand-primary: rgb(14, 120, 219); --brand-primary-hover: oklch(from var(--brand-primary) calc(l * .90) c h); --brand-primary-active: oklch(from var(--brand-primary) calc(l * .80) c h); --brand-primary-opacity-4: rgba(14, 120, 219, 0.04); --brand-primary-opacity-10: rgba(14, 120, 219, 0.10); --brand-primary-opacity-20: rgba(14, 120, 219, 0.20); --brand-primary-opacity-40: rgba(14, 120, 219, 0.40); /* Gradients brand (source) */ --brand-gradient-surface-primary: linear-gradient( to bottom, oklch(from var(--brand-primary) calc(l * 1.10) c h), var(--brand-primary) ); --brand-gradient-surface-overlay: linear-gradient( to bottom, oklch(from var(--brand-primary) calc(l * 0.88) c h / 0.95), oklch(from var(--brand-primary) calc(l * 0.42) c h / 0.98) ); --brand-gradient-text-primary: linear-gradient( 135deg, var(--brand-primary-active), var(--brand-primary) ); }

Toutes les valeurs utilisées par les utilitaires et composants. Ce bloc est theme-aware.

:root, [data-scheme] { /* Primary */ --color-primary: var(--brand-primary); --color-primary-hover: var(--brand-primary-hover); --color-primary-active: var(--brand-primary-active); --color-primary-opacity-4: var(--brand-primary-opacity-4); --color-primary-opacity-10: var(--brand-primary-opacity-10); --color-primary-opacity-20: var(--brand-primary-opacity-20); --color-primary-opacity-40: var(--brand-primary-opacity-40); /* Borders (theme-aware) */ --color-border-primary: var(--brand-primary); --color-border-primary-20: var(--brand-primary-opacity-20); /* Surfaces (alias) */ --surface-page: var(--color-background); --surface-container: var(--color-container); --surface-overlay: var(--color-overlay); /* Gradients remappés */ --gradient-surface-primary: var(--brand-gradient-surface-primary); --gradient-surface-overlay: var(--brand-gradient-surface-overlay); --text-gradient: var(--brand-gradient-text-primary); --gradient-primary: var(--gradient-surface-primary); /* Gradient secondaire basé sur les fonds */ --gradient-secondary: linear-gradient(to bottom, var(--color-background), var(--color-container)); }

Tip : Branchez vos composants sur --surface-* (et non des couleurs en dur).
