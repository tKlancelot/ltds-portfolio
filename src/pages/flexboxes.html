Flexbox

Utilitaires Flexbox pour gérer disposition, alignements et responsive. Ils couvrent display, flex-direction, wrap, justify-content, align-*, grow/shrink, basis, et order. Des variantes responsive sont générées pour les breakpoints (desk, lap, note, tab, mob, smob).

.u-flex { display: flex; } .u-inline-flex { display: inline-flex; }

Définir l’axe principal : horizontal/vertical, normal ou inversé.

.u-flex-row { flex-direction: row; } .u-flex-row-rev { flex-direction: row-reverse; } .u-flex-col { flex-direction: column; } .u-flex-col-rev { flex-direction: column-reverse; } .u-flex-nowrap { flex-wrap: nowrap; } .u-flex-wrap { flex-wrap: wrap; } .u-flex-wrap-rev { flex-wrap: wrap-reverse; } .u-justify-start { justify-content: flex-start; } .u-justify-center { justify-content: center; } .u-justify-end { justify-content: flex-end; } .u-justify-between { justify-content: space-between; } .u-justify-around { justify-content: space-around; } .u-justify-evenly { justify-content: space-evenly; } .u-items-start { align-items: flex-start; } .u-items-center { align-items: center; } .u-items-end { align-items: flex-end; } .u-items-stretch { align-items: stretch; } .u-items-baseline{ align-items: baseline; } .u-content-start { align-content: flex-start; } .u-content-center { align-content: center; } .u-content-end { align-content: flex-end; } .u-content-between { align-content: space-between; } .u-content-around { align-content: space-around; } .u-content-evenly { align-content: space-evenly; } .u-content-stretch { align-content: stretch; } .u-self-auto { align-self: auto; } .u-self-start { align-self: flex-start; } .u-self-center { align-self: center; } .u-self-end { align-self: flex-end; } .u-self-stretch { align-self: stretch; } .u-self-baseline { align-self: baseline; } .u-grow { flex-grow: 1; } .u-grow-0 { flex-grow: 0; } .u-shrink { flex-shrink: 1; } .u-shrink-0 { flex-shrink: 0; } .u-basis-auto { flex-basis: auto; } .u-basis-0 { flex-basis: 0; } .u-basis-1-2 { flex-basis: 50%; } .u-basis-1-3 { flex-basis: 33.3333%; } .u-basis-2-3 { flex-basis: 66.6667%; } .u-basis-1-4 { flex-basis: 25%; } .u-basis-3-4 { flex-basis: 75%; }

De .u-order-0 à .u-order-6.

.u-order-0 { order: 0; } .u-order-1 { order: 1; } ... .u-order-6 { order: 6; }

Pour chaque breakpoint max-width (desk, lap, note, tab, mob, smob), les variantes essentielles sont générées : display, direction (row/col), justify, items, content, grow/shrink.

/* Exemple pour tab (≤1024px) */ @media (max-width: 1024px) { .u-tab-flex { display: flex; } .u-tab-flex-row { flex-direction: row; } .u-tab-flex-col { flex-direction: column; } .u-tab-justify-between { justify-content: space-between; } .u-tab-items-center { align-items: center; } .u-tab-content-start { align-content: flex-start; } .u-tab-grow { flex-grow: 1; } .u-tab-shrink-0 { flex-shrink: 0; } }
