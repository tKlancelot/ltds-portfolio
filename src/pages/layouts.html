Layouts

LTDS Layout System propose deux primitives ultra-légères : lt-frame (pile verticale) et lt-stack (rangée horizontale). Tout est piloté par des variables CSS et des modificateurs, avec variantes max-width : sm, mob, tab, note, lap, desk.

Par défaut : lt-frame empile en colonne, lt-stack aligne en rangée. Les gaps/alignements/justifies sont contrôlés via des tokens et des classes utilitaires.

/* Valeurs par défaut */ .lt-frame { --frame-gap: var(--size-4); --frame-align: stretch; --frame-justify: flex-start; display: flex; flex-direction: column; gap: var(--frame-gap); align-items: var(--frame-align); justify-content: var(--frame-justify); } .lt-stack { --stack-gap: var(--size-4); --stack-align: center; --stack-justify: flex-start; --stack-wrap: wrap; display: flex; flex-direction: row; gap: var(--stack-gap); align-items: var(--stack-align); justify-content: var(--stack-justify); flex-wrap: var(--stack-wrap); }
