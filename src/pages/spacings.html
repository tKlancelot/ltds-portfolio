Spacings

Utilitaires d’espacement générés depuis les tokens --size-* et --gap-*. Couvre gap (Grid/Flex), padding (all/axes/côtés + responsive), margin (positifs & négatifs) et quelques raccourcis.

Contrôle de l’espace entre items pour Grid et Flex.

.u-gap-0 { gap: 0; } .u-gap-1 { gap: var(--size-1); } ... .u-gap-12 { gap: var(--large-size-2); } /* 120px */

Renseigne rapidement les espacements internes d’un bloc.

/* tout / axes */ .u-p-4 { padding: var(--size-4); } .u-px-3 { padding-inline: var(--size-3); } .u-py-6 { padding-block: var(--size-6); } /* côtés */ .u-pt-2 { padding-top: var(--size-2); } .u-pr-2 { padding-right: var(--size-2); } .u-pb-2 { padding-bottom: var(--size-2); } .u-pl-2 { padding-left: var(--size-2); }

Variante par palier : smob ≤414, mob ≤768, tab ≤1024, lap ≤1439, desk ≤1919.

/* exemples */ .u-mob-p-3 { padding: var(--size-3); } .u-tab-px-6 { padding-inline: var(--size-6); } .u-lap-pt-8 { padding-top: var(--size-8); }

Spacings externes d’un élément.

.u-m-4 { margin: var(--size-4); } .u-mx-2 { margin-inline: var(--size-2); } .u-my-6 { margin-block: var(--size-6); } .u-mt-1 { margin-top: var(--size-1); } .u-mr-1 { margin-right: var(--size-1); } .u-mb-1 { margin-bottom: var(--size-1); } .u-ml-1 { margin-left: var(--size-1); }
.u-m-2
.u-mt-4 + .u-mb-4

Utile pour compenser des paddings ou créer des effets d’emprise. (Non générés pour l’indice 0)

.u--m-2 { margin: calc(var(--size-2) * -1); } .u--mx-3 { margin-inline: calc(var(--size-3) * -1); } .u--mt-1 { margin-top: calc(var(--size-1) * -1); }
Parent padding 4
Child compensé (.u--mx-4)

Raccourcis fréquents pour centrer / laisser le navigateur allouer l’espace.

.u-m-auto { margin: auto; } .u-mx-auto { margin-inline: auto; } .u-my-auto { margin-block: auto; }
Bloc centré horizontalement
  • Favorisez gap sur le parent plutôt que des marges empilées sur les enfants.
  • Restez cohérent avec l’échelle : utilisez les indices (1..10 puis 11..12 pour large) plutôt que des valeurs ad hoc.
  • N’utilisez les marges négatives que pour des cas précis (compensation de container, effets de débordement maîtrisés).
